# IntegrativeLearningExperience
Integrative Learning Experience from the Moldova Dataset 
In this project I will be creating the R code for my integrative learning experience at
Boston University in the Epidemiology and Biostatistics Department. 
I will be using the "Moldova" dataset, which examines tuberculosis data from Moldova collected
in 2007-2009
The question I wish to answer is "Is having been incarcerated at any point associated with the development of multi-drug resistant tuberculosis (TB) infection, among TB positive adults in Moldova from 2007-2010? and is this relationship different by HIV Co infection?"
